## Check Time Driff
   *❔ Question*: Trên Windows server, làm thế nào để check Time Driff với một reference ntp?  
   🚩 Answer: Run below command
   ```
   w32tm /stripchart /computer:us.pool.ntp.org /dataonly /samples:5
   ```
